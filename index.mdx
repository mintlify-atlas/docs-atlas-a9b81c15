  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Generador QR Pro in just a few steps
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Generador QR Pro repository to your local machine:
      
      ```bash
      git clone https://github.com/WaltherFgs/generador_qr.git
      cd generador_qr
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install the required npm packages:
      
      ```bash
      npm install
      ```
      
      The project uses React 19, Vite, qrcode.react, and lucide-react for icons.
    </Step>
    
    <Step title="Start the development server">
      Launch the local development server:
      
      ```bash
      npm run dev
      ```
      
      Open your browser to `http://localhost:5173` to see the QR generator in action.
    </Step>
    
    <Step title="Create your first QR code">
      Select a template (URL, WiFi, vCard, or Email), enter your data, customize the colors and styling, then export as PNG or SVG.
    </Step>
  </Steps>
</div>
